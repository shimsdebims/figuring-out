## Web App
CropGuard - AI Plant Disease Detection

CropGuard is an AI-powered plant disease detection application using MobileNetV2 trained on 54,000+ PlantVillage images. The app analyzes uploaded images and provides instant diagnosis with treatment recommendations for 38 different plant diseases.

Access the website: https://figuring-out-advdrzyhouwi2axrrwgstu.streamlit.app/
